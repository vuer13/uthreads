# uthreads

A small cooperative user-level threading library in C

## Features
- FIFO Cooperative Scheduling
- User-level thread creation
- Thread yield, exit, join, detach

## Build
``` bash
make
```

## Run example
``` bash
make
```

## Current limitations
- No priority scheduling yet
- No mutexes yet
- No condition variables yet
- No preemption
- No timers
